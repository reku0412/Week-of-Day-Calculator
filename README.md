# 📅 Date Utility

A web application that analyzes a selected date and displays useful information.

🌐 **Live Demo**  
https://week-of-day-calculator.onrender.com

---

## ✨ Features

- 🗓️ Calculate the day of the week
- 🌏 Determine whether the year is a leap year
- 📆 Display the day number within the year
- ⏳ Show the remaining days until the end of the year
- ⚠️ Validate invalid dates
- 💻 Responsive web interface

---

## 🛠️ Built With

- Python 3
- Flask
- HTML5
- CSS3
- Git & GitHub
- Render
